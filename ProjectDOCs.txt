1) Why git ? :

	You can keep track of changes
	You can merge a team of developers to work on a single projegt without intterruptin each other for vertion control  ( working independently )
	If the changes your team make are not desired, you can simply roll back to previous versions and continue development from there.
	It'll be unessessery to communincate with a central server for data exchange or developing the softwere ( by using desributed server )
	








2) How to install git :
	
	For linux systems, you use the terminal to get and install the app by apt-get install, pacman -s  , etc...
	For windows systems you simply download the setup wizard from a trusted source and you install it like a normal app.
